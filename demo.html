Notes and Options

Calling the .unmask() method on a jquery collection will "unmask" the first member of that collection, by layering masks above the rest of the viewport.

This plugin is intended as an alternative to those like jQueryTools' .expose(), in that it still works even if the target element has parent elements with z-indeces that may interfere with stacking above the mask. Because it does not layer an element above the target, click and hover interactions are not affected.

Clear the masks by clicking or via the ESC key, if allowed in the options passed (both available by default, see below). If you call unmask() on an element without clearing an existing visible mask, unmask will just move the clear region to cover the new element, with animation if allowed.

Options:

The following are the defaults for the options object passed when unmasking an element.

allowAnimations: true,  // animate transitions between unmask targets while mask remains up
bgColor: '',            // override CSS-defined mask color -- CSS value is rgba(0, 0, 0, 0.66)
closeOnClick : true,    // click on the mask to clear masks
closeOnEsc : true       // press ESC key to clear masks
